파이썬 웹개발 요리책

* 모듈

## 표준 라이브러리들

중요한 표준 라이브러리는 잘 익혀둡시다. 반드시 필요할 날이 올거에요.

* os.path: 파일 경로 다루기
* datetime: 날짜와 시간 처리
* json: JSON 파서

## 여러가지 라이브러리들

쓸만한 라이브러리를 모아봤습니다.

* PIL: 이미지 처리
* Requests: HTTP 통신
* lmxl: XML, HTML 파서
* Celery: 분산 작업 처리
* Babel: 다국어 지원과 국제화
